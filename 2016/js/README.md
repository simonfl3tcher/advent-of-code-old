# 2016-advent-of-code

My code for the 2016 Advent Of Code challenge.

## Find answer to single a day

  1. cd into the day:
  ```bash
  cd day-1
  ```

  2. Run node
  ```bash
  node
  ```

  3. Load file into REPL
  ```javascript
  .load solve.js
  ```

  4. Run module
  ```javascript
  module.run()
  ```
